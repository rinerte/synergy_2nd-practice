    1. Установите git с официального сайте если у вас его ещё нет: https://git-scm.com/install/
    2. Скопируйте репозиторий с программой в папку на компьютере с помощью следующей команды: 
       git clone https://github.com/rinerte/synergy_2nd-practice.git
    3. Установите python 3.12 с официального сайта: https://www.python.org/downloads/. Учтите, что библиотека spaCy – пока что не работает на версии выше 3.12
    4. Откройте папку с проектом и установите зависимости с помощью следующей команды:
       py -m pip install -r requirements.txt
    5. Запустите программму
       py ./bot.py
